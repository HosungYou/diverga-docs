## Key Features

### Interactive Elements
1. **Progress Indicator** - Numbered circles (1-6) at top, clickable anchors
2. **Copy Buttons** - Every code block has copy-to-clipboard
3. **Color Coding** - Each step has unique color from void palette
4. **Animations** - Fade-in on scroll using Framer Motion
5. **Hover Effects** - Cards glow on hover

### Content Sections Per Step
- **Icon + Number** - Visual step identifier
- **Title** - Clear step name
- **Checklist** - Bullet points with chevron icons
- **Code Block** - Syntax highlighted with copy button
- **Output Example** - Expected terminal output (Steps 3-4)
- **Explanation Box** - Additional context (Step 5)
- **Link Cards** - Navigation to related docs (Step 6)

### Typography
- **Headings**: font-display class (custom font)
- **Code**: font-mono class
- **Body**: Default system fonts
- **Colors**: stellar-bright, stellar-dim, stellar-faint

### Responsive Design
- **Mobile**: Single column, stacked steps
- **Tablet**: Optimized spacing
- **Desktop**: Full width with proper margins
